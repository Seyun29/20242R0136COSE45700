### Project Overview
- Vector Graphic Editor developed with Java Swing GUI
- Features
  - Draw several types of objects on canvas
    - Image, Text, Line, Rectangle, Ellipse
  - Resize, Move, Update, Adjust Z-Order of Objects
  - Multi Object Selection
